Two Sum

HashMap Lookup
โจทย์
ให้ array ของตัวเลข และ target — หา index ของเลข 2 ตัวที่บวกกันแล้วได้ target
ตัวอย่าง
Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1] → เพราะ nums[0] + nums[1] = 2 + 7 = 9
วิธีคิด
  • สร้าง map ว่าง (seen)
  • วนทุกตัว คำนวณ need = target - num
  • ถ้า need อยู่ใน map → เจอคำตอบแล้ว!
  • ถ้าไม่มี → เก็บ seen[num] = i แล้ววนต่อ
    • โค้ด
    func twoSum(nums []int, target int) []int {
    seen := map[int]int{}

    for i, num := range nums {
        need := target - num

        index, exists := seen[need]
        if exists {
            return []int{index, i}
        }
        seen[num] = i
    }
    return []int{}
}
    Complexity
    Time
    O(n)
    Space
    O(n)
    เช็คก่อน เก็บทีหลัง — pattern นี้ใช้บ่อยมาก!

    Valid Anagram

    HashMap Count
    โจทย์
    เช็คว่า string 2 ตัวเป็น Anagram กันไหม (ใช้ตัวอักษรเหมือนกันทุกตัว แค่สลับที่)
    ตัวอย่าง
    "anagram" และ "nagaram" → true
    "rat" และ "car" → false
    วิธีคิด
  • วน s → เพิ่มนับ seen[ch]++
  • วน t → ลดนับ seen[ch]--
  • ถ้าทุก count เป็น 0 = anagram
    • โค้ด
    func isAnagram(s string, t string) bool {
    seen := map[rune]int{}

    for _, ch := range s { seen[ch]++ }
    for _, ch := range t { seen[ch]-- }

    for _, count := range seen {
        if count != 0 { return false }
    }
    return true
}
    Complexity
    Time
    O(n)
    Space
    O(n)
    บวกจาก s ลบจาก t — ถ้าทุกช่องเป็น 0 แสดงว่าสมดุล = anagram

    Contains Duplicate

    HashMap Lookup
    โจทย์
    เช็คว่า array มีเลขซ้ำกันไหม — ถ้ามีซ้ำ return true
    ตัวอย่าง
    [1, 2, 3, 1] → true
    [1, 2, 3, 4] → false
    วิธีคิด
  • ใช้ map[int]bool เป็นสมุดจด
  • ถ้าเจอเลขที่อยู่ใน map แล้ว → ซ้ำ! return true
  • ถ้ายังไม่เคยเจอ → จดลง map
    • โค้ด
    func containsDuplicate(nums []int) bool {
    seen := map[int]bool{}

    for _, num := range nums {
        if seen[num] {
            return true
        }
        seen[num] = true
    }
    return false
}
    Complexity
    Time
    O(n)
    Space
    O(n)
    map[int]bool ใช้ค่า false เป็น default — ถ้ายังไม่เคยใส่ seen[num] จะ return false อัตโนมัติ

    Best Time to Buy and Sell Stock

    Greedy / 1-Pass
    โจทย์
    ให้ array ราคาหุ้นแต่ละวัน — หากำไรสูงสุดที่ทำได้ (ซื้อก่อนขายทีหลัง)
    ตัวอย่าง
    [7, 1, 5, 3, 6, 4] → 5 (ซื้อวันที่ 2 ราคา 1 ขายวันที่ 5 ราคา 6)
    [7, 6, 4, 3, 1] → 0 (ราคาลงตลอด ไม่ซื้อ)
    วิธีคิด
  • จำ minPrice (ราคาต่ำสุดที่เคยเห็น) ไว้ตลอด
  • ทุกวัน คำนวณกำไรถ้าขายวันนี้ = price - minPrice
  • อัปเดต maxProfit ถ้ากำไรวันนี้ดีกว่า
    • โค้ด
    func maxProfit(prices []int) int {
    minPrice := prices[0]
    maxProfit := 0

    for _, price := range prices {
        if price < minPrice {
            minPrice = price
        }
        if price-minPrice > maxProfit {
            maxProfit = price - minPrice
        }
    }
    return maxProfit
}
    Complexity
    Time
    O(n)
    Space
    O(1)
    วนรอบเดียว จำแค่ 2 ค่า — minPrice กับ maxProfit ไม่ต้องใช้ map เลย

    Reverse String

    Two Pointers
    โจทย์
    กลับ []byte ในตัวมันเอง (in-place) ไม่ต้อง return
    ตัวอย่าง
    "hello" → "olleh"
    "Go" → "oG"
    วิธีคิด
  • ใช้ pointer 2 ตัว — left (ซ้าย) และ right (ขวา)
  • สลับ s[left] กับ s[right]
  • เลื่อน left++ และ right-- จนชนกัน
    • โค้ด
    func reverseString(s []byte) {
    left := 0
    right := len(s) - 1

    for left < right {
        s[left], s[right] = s[right], s[left]
        left++
        right--
    }
}
    Complexity
    Time
    O(n)
    Space
    O(1)
    Two Pointers สลับในตัว ไม่ต้องสร้าง array ใหม่ — ประหยัด memory

    First Unique Character

    HashMap Count
    โจทย์
    หา index ของตัวอักษรตัวแรกที่ไม่ซ้ำใน string ถ้าไม่มีเลย return -1
    ตัวอย่าง
    "leetcode" → 0 (l ไม่ซ้ำ และอยู่ตำแหน่งแรกสุด)
    "loveleetcode" → 2 (v)
    "aabb" → -1
    วิธีคิด
  • รอบ 1: วน s นับจำนวนแต่ละตัวอักษร count[ch]++
  • รอบ 2: วน s อีกครั้ง หาตัวแรกที่ count[ch] == 1
  • return index นั้น ถ้าไม่เจอ return -1
    • โค้ด
    func firstUniqChar(s string) int {
    count := map[rune]int{}

    for _, ch := range s { count[ch]++ }

    for i, ch := range s {
        if count[ch] == 1 { return i }
    }
    return -1
}
    Complexity
    Time
    O(n)
    Space
    O(n)
    วน 2 รอบ — รอบแรกนับ รอบสองหา ต้องรักษาลำดับจึงวน string ใหม่

    Palindrome Number

    Math / Modulo
    โจทย์
    เช็คว่าตัวเลขอ่านหน้า-หลังเหมือนกันไหม (โดยไม่แปลงเป็น string)
    ตัวอย่าง
    121 → true
    -121 → false (เลขติดลบ)
    10 → false
    วิธีคิด
  • เลขติดลบ → false ได้เลย
  • ดึงหลักสุดท้าย: digit = x % 10
  • สร้างเลขกลับด้าน: reversed = reversed*10 + digit
  • ตัดหลักสุดท้ายทิ้ง: x = x / 10
  • เทียบ original กับ reversed
    • โค้ด
    func isPalindrome(x int) bool {
    if x < 0 { return false }

    original := x
    reversed := 0

    for x > 0 {
        digit := x % 10
        reversed = reversed*10 + digit
        x = x / 10
    }
    return original == reversed
}
    Complexity
    Time
    O(log n)
    Space
    O(1)
    % 10 ดึงหลักสุดท้าย, / 10 ตัดหลักออก — สร้างเลขกลับด้านด้วยเลขคณิตล้วนๆ

    Fibonacci Number

    Loop DP
    โจทย์
    คืนค่า Fibonacci ลำดับที่ n — F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2)
    ตัวอย่าง
    fib(4) → 3 (0, 1, 1, 2, 3)
    fib(10) → 55
    วิธีคิด
  • จำแค่ 2 ค่าล่าสุด a และ b
  • ทุกรอบ: a, b = b, a+b
  • วน n รอบ → a คือคำตอบ
    • โค้ด
    func fib(n int) int {
    a, b := 0, 1

    for i := 0; i < n; i++ {
        a, b = b, a+b
    }
    return a
}
    Complexity
    Time
    O(n)
    Space
    O(1)
    a, b = b, a+b — Go swap พร้อมกันในบรรทัดเดียว ไม่ต้องใช้ตัวแปรชั่วคราว

    FizzBuzz

    If/Else + Modulo
    โจทย์
    วน 1 ถึง n — หาร 3 → "Fizz", หาร 5 → "Buzz", หาร 15 → "FizzBuzz", นอกนั้น → ตัวเลข
    ตัวอย่าง
    n=5 → ["1", "2", "Fizz", "4", "Buzz"]
    n=15 → [..., "FizzBuzz"]
    วิธีคิด
  • เช็ค % 15 ก่อนเสมอ (FizzBuzz ต้องมาก่อน)
  • แล้วค่อย % 3, % 5, และ default
  • append เข้า []string แล้ว return
    • โค้ด
    func fizzBuzz(n int) []string {
    result := []string{}

    for i := 0; i < n; i++ {
        if (i+1)%15 == 0 {
            result = append(result, "FizzBuzz")
        } else if (i+1)%3 == 0 {
            result = append(result, "Fizz")
        } else if (i+1)%5 == 0 {
            result = append(result, "Buzz")
        } else {
            result = append(result, strconv.Itoa(i+1))
        }
    }
    return result
}
    Complexity
    Time
    O(n)
    Space
    O(n)
    ลำดับ if สำคัญมาก — FizzBuzz ต้องเช็คก่อน ไม่งั้นถูก Fizz หรือ Buzz ดักไว้

    Count Primes

    Helper Function
    โจทย์
    นับจำนวน prime number ที่น้อยกว่า n
    ตัวอย่าง
    n=10 → 4 (2, 3, 5, 7)
    วิธีคิด
  • เขียน function isPrime(n) แยกออกมา
  • isPrime: วน j จาก 2 ถึง n-1 ถ้าหารลงตัว = ไม่ใช่ prime
  • countPrimes: วน i แล้วเรียก isPrime(i) นับ count
    • โค้ด
    func isPrime(n int) bool {
    for j := 2; j < n; j++ {
        if n%j == 0 { return false }
    }
    return true
}

func countPrimes(n int) int {
    count := 0
    for i := 2; i < n; i++ {
        if isPrime(i) { count++ }
    }
    return count
}
    Complexity
    Time
    O(n√n)
    Space
    O(1)
    แยก helper function ออกมาทำให้โค้ดอ่านง่าย — แนวคิดสำคัญในการเขียนโค้ดที่ดี
